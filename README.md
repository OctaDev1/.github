# 🚀 OctaDev

### Transformando ideias em soluções através da tecnologia.

Equipe de desenvolvimento formada durante o **Bootcamp Java Full Stack da Generation Brasil**, dedicada à criação de projetos utilizando metodologias ágeis, boas práticas de programação e tecnologias modernas.

</div>

---

## 💡 Sobre a Equipe

A **OctaDev** é composta por oito desenvolvedores apaixonados por tecnologia e desenvolvimento de software.

Nosso objetivo é criar aplicações escaláveis, organizadas e de qualidade, aplicando conhecimentos em desenvolvimento Full Stack, arquitetura de software, bancos de dados e metodologias ágeis.
